# Pardes

Silly but cute new tab extension for Chrome :3

## Features

- **Clock** - digital or analog, 12h/24h, toggle seconds
- **Date** - current date + monthly calendar, expandable
- **Todo** - add tasks with optional dates, mark done, delete
- **Favorites** - 3 speed-dial slots with favicon auto-fetch
- **Background** - upload an image/video or paste a URL, adjust blur, brightness, rotate, and reposition the visible area
- **Customize panel** - show/hide widgets, reorder by drag-and-drop, adjust glass opacity, toggle sidebar mode
- **Persistence** - all settings stored in localStorage and background blobs in IndexedDB

## Install

1. Open `chrome://extensions`
2. Enable Developer mode
3. Click "Load unpacked" and select this directory
